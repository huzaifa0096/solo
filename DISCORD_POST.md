## Version B (with the meta-narrative — also strong)

> Week 4 submission: **Solo** — an AI-owned business on Locus 🟢
>
> The pitch: every demo I've seen of agentic payments shows one side. Agent receives money OR agent spends money. Locus's bet is the policy layer that lets you safely close both — and I wanted to build the smallest possible proof of that thesis.
>
> So Solo is a SaaS where the CEO is Claude. Customers pay USDC, the agent fulfils with Anthropic, settles the API cost back through Locus, and the daily cap + vendor allowlist + per-order limit run on *every* spend.
>
> Hero feature: when an order is fulfilling, the page streams every agent decision live — "estimating cost", "running Locus policy check", "vendor=anthropic on allowlist", "settling $0.31 via Locus", "delivered". This is what trustworthy agent execution looks like.
